Semua data password di-encrypt dengan AES-256-GCM dan disimpan hanya di device kamu. Key enkripsi di-derive dari master password pakai Argon2id — artinya tanpa master password kamu, bahkan kami pun tidak bisa buka vault-nya. Zero-knowledge by design.
Apa itu One-Time Payment?
Kamu bayar sekali $9.99 dan dapat semua fitur Premium selamanya — termasuk update gratis untuk versi berikutnya. Tidak ada tagihan bulanan, tidak ada auto-renew. Kebalikan dari semua password manager berbasis langganan.
Bisa digunakan offline?
Ya. Hesych 100% berjalan lokal di device kamu — tidak butuh koneksi internet untuk buka vault, generate password, atau isi form otomatis. Cloud sync di versi Premium bersifat opsional dan bisa dimatikan kapan saja.
